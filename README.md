# SQL-Driven-Sales-Analysis-for-a-Digital-Music-Store

## Project Title

Music Store Sales Analysis Using SQL

## Business Problem

The business wanted to understand customer purchasing behavior, genre performance, and sales trends to improve marketing and retention strategies.

## Objectives

Identify top-performing music genres

Analyze weekly sales trends for Rock music

Measure average transaction value

Identify repeat customers

## Tools Used

SQL (SQLite)

Excel / Tableau (for visualization)

GitHub (documentation)

## Key Insights

Rock music is the top-selling genre with 835 purchases

Sales peak on Thursdays and Mondays

Average Rock transaction value is $9.25

A small group of customers accounts for a high number of repeat purchases

## Business Recommendations

Prioritize Rock music in promotions

Launch weekday-focused campaigns

Introduce loyalty incentives for high-value customers
